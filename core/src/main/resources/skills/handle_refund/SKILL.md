# 退款处理流程

这是一个**纯文字技能**：没有脚本，正文是给你（Agent）的一套操作流程。被调用时你会拿到这段步骤，
然后用**现有工具**按规矩办，不要自己写代码。

1. 用 `get_balance` 查该玩家当前余额，确认账目对得上。
2. 用 `knowledge_search` 查「退款政策」，对照判断本次申诉是否符合。
3. **符合** → 用 `run_command_on_server` 执行 `eco take <玩家> <金额>`，并私聊告知玩家已处理。
4. **不符合** → 礼貌说明原因，**不执行任何扣减**。
5. 无论结果，用 `remember` 记一笔本次处理（玩家、金额、结论），便于后续追溯。

⚠️ 单笔金额 > 1000 必须走人工审批，禁止自动执行扣减。
