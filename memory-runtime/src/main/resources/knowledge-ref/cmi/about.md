## Welcome to the CMI plugin

The CMI plugin is made for Spigot and Paper server versions 1.7.x and up, and helps server owners conveniently manage their servers with an extensive feature set. It has a flexible and dynamic set of commands, permissions, and placeholders. CMI offers your players a modern and consistent experience. 

This page introduces you to a lot of the features such as BungeeCord, Economy Engine, Chat Manager, and more. Our top menu will have links to pages that dive into the specifics for each feature. If you’re a bit overwhelmed by the 300 commands and features, feel free to join our Discord server where community members can help you answer some of your questions.

This is a premium plugin, it requires the free CMILib library to run, and if you have any pre-sales questions there’s an FAQ (top right in the) menu that will hopefully answer any question you might have prior to purchase. 

**Run a network of servers?** The free **[CMI Bungee](https://www.zrips.net/cmib/)** or [**CMI Velocity**](https://www.zrips.net/cmiv/) plugin is recommended for you to utilize all supported capabilities of CMI plugin while running it at BungeeCord network. Without it, some features will have limited capabilities. Download it and put into BungeeCord/Velocity servers plugin folder and reload it. 

**Economy**: For CMI economy to properly work you will need one of two solutions:   
1. Is to use recompiled Vault version from [**HERE**](https://www.zrips.net/cmivault/) which has direct support for CMI economy and will have the best compatibility with other plugins.   
2. Use the economy injector from [**HERE**](https://zrips.net/cmii/) which will add CMI economy support to any Vault build.  
  
Both methods are valid but the second one can have small issues with plugins which would tend to load straight after Vault loads and before the injector himself.   
P.S. Don’t forget to enable Economy in config file.

This plugin is made to be **all in one** plugin for server management and replace multiple outdated plugins. It can **fully replace** Essentials/EssentialsX and contains simple conversion commands to transfer old player data to new system. It offers more features than you expect and provides most needed tools for creating your own unique server with new game mechanics.  
Features ranging from basic ones, like heal, to most advanced ones, like kit editors ingame, item finder across all possible containers, portal systems, self updating signs, clickable holograms and so on.

​For developers who want to use **CMI** as a dependency for particular features and need to have API to work with. You can download the clean version of the plugin from [**HERE**](https://www.zrips.net/wp-content/uploads/2021/09/CMI9.0.0.0API.jar). This version contains all methods used in the original plugin but is stripped down from any actual code, so the plugin will not work on the actual server but will help to develop a plugin of your own based on the CMI engine. Github repository can be found [**HERE**](https://github.com/Zrips/CMI-API)

![](https://www.zrips.net/wp-content/uploads/2019/02/localeFiles.jpg)

All messages can be translated and separate locale files can be created for easy use. By default, plugin is translated to EN language.

Some extra languages are included. They cannot be fully translated, any contribution for additional translation are welcome ^.^ :

-   **Spanish**
-   **Chinese**
-   **German**
-   **Italian**
-   **Russian**
-   **and more! [Complete list on GitHub](https://github.com/Zrips/CMI/tree/master/Translations).**

**Setup:**

1.  Drag and Drop jar into your plugins folder
    
2.  Start server for default files to be created
    
3.  Edit by your choice and enjoy ![:)](https://www.spigotmc.org/styles/default/xenforo/clear.png "Smile :)")
