# /dh displays swap-lines

## Swaps two lines in a Text Display Entity

The `swap-lines` subcommand switches the position of the first specified line with the second one.

This command only works for Text Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.text.swaplines`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `swaplines`

## Arguments

### `<name>`

Name of the Display entity to swap lines in.

### `<line1>`

The first line to swap.

### `<line2>`

The second line to swap.
