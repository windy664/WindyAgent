# /dh displays removeline

## Removes a Line from a Text Display Entity

The `removeline` subcommand removes a line from a Display Entity.  
Any lines after the specified one are moved up by one.

This command only works for Text Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.text.removeline`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `remline`

## Arguments

### `<name>`

Name of the Display entity to remove a line from.

### `<index>`

Position of the line to remove.
