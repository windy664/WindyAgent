# /dh displays delete

## Deletes a Display Entity

The `delete` subcommand removes a Display Entity.  
This **permanently** removes the Display Entity, including its configuration file.

To only hide a Display Entity, use [[/dh displays disable|`/dh displays disable`]].

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.delete`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `del`

## Arguments

### `<name>`

Name of the Display entity to remove.
