# /dh displays list-attributes

## Lists all configured Attributes for a Display Entity

The `list-attributes` subcommand will display all currently configured attributes for the provided Display Entity.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.listattributes`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `listattributes`

## Arguments

### `<name>`

Name of the Display entity to lisst attributes of.
