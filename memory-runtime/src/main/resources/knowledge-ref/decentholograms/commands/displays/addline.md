# /dh displays addline

## Adds a Line to a Text Display Entity

The `addline` subcommand adds a new Line to a Display Entity.

This command only works for Text Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.text.addline`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `appendline`

## Arguments

### `<name>`

Name of the Display entity to add a new line to.

### `<text>`

New Text line to add.
