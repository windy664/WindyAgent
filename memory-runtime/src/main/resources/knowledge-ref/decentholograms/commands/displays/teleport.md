# /dh displays teleport

## Teleports you to the Display Entity

This command is only usable by Players!

The `teleport` subcommand teleports you to the specified Display Entity's location.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.teleport`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `tp`
-   `tele`

## Arguments

### `<name>`

Name of the Display entity to teleport to.
