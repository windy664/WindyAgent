# /dh displays display-range

## Sets the range a Display Entity should be visible from

The `display-range` subcommand sets the max range a player can be from the Display Entity to still see it.

The client's and server's entity render distance may influence this.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.displayrange`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `displayrange`

## Arguments

### `<name>`

Name of the Display entity to set the display range of.

### `<range>`

The range in blocks to set.  
Allowed values are between 1 and 256.
