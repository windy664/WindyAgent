# /dh displays center

## Centers the Display Entity

The `center` subcommand centers the provided display Entity to the closest block.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.center`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to center.
