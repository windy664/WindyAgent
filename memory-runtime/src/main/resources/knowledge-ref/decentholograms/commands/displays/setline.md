# /dh displays setline

## Sets a line in the Text Display Entity

The `setline` subcommand sets the content of the specified line in a Display Entity.

This command only works for Text Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.text.setline`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to set the line of.

### `<index>`

Position of the Line to edit.

### `<text>`

New text for the line to have.
