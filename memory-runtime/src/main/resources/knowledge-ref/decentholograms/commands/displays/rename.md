# /dh displays rename

## Renames a Display Entity

The `rename` subcommand renames the Display Entity to the new name.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.rename`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to rename.

### `<new_name>`

The new name for the Display Entity.
