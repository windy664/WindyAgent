# /dh displays nearby

## Lists all nearby Displa Entities

This command is only usable by a Player!

The `nearby` subcommand lists all Display Entities that are within the specified radius of your location.  
Each entry contains the Display Entity's Name and Location.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.nearby`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `near`

## Arguments

### `<range>`

Radius around yourself to look for Display Entities.

### `[page]`

Optional page number to show the page of in the list.
