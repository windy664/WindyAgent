# /dh displays list

## Lists all currently created Display Entities

This command is only usable by Players!

The `list` subcommand lists all currently existing Display Entities.  
Each entry contains the Display's name and location.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.list`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `[page]`

Page number in the list to show.
