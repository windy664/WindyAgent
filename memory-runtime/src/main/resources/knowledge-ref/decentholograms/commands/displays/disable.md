# /dh displays disable

## Disables a Display Entity

The `disable` subcommand disables a Display Entity.  
A disabled Display Entity won't be visible for any player, but still exists as a file in the server to [[/dh displays enable|re-enable]].

To fully remove a Display Entity, use [[/dh displays delete|`/dh displays delete`]].

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.disable`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to disable.
