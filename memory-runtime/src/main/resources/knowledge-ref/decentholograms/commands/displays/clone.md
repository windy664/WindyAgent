# /dh displays clone

## Clones a Display Entity

The `clone` subcommand clones the provided Display Entity.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.clone`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to clone.

### `<new_name>`

New name for the copy to have.
