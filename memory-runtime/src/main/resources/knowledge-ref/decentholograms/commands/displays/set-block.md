# /dh displays set-block

## Set the displayed block in a Block Display

The `set-block` subcommand sets the block to display in an Display entity.

This command only works for Block Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.setblock`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `setblock`
-   `block`

## Arguments

### `<name>`

Name of the Display entity to change the block of.

### `<block_type>`

The Block to set. Needs to be a valid Block ID in the format `namespace:id` (i.e. `minecraft:dirt`).
