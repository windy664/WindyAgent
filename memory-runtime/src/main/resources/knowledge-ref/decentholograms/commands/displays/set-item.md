# /dh displays set-item

## Sets the item to display

The `set-item` subcommand sets the Item to display for the Display Entity.

This command only works for Item Display Entities!

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.setitem`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `setitem`
-   `item`

## Arguments

### `<name>`

Name of the Display entity to set the item of.

### `<item>`

The Item to set. Needs to be a valid Item ID in the format `namespace:id` (i.e. `minecraft:dirt`).
