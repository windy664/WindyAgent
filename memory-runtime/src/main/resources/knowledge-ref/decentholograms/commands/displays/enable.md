# /dh displays enable

## Enables a Display Entity

The `enable` subcommand enables a Display Entity.  
Enabled display Entities are visible for all players.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.enable`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<name>`

Name of the Display entity to enable.
