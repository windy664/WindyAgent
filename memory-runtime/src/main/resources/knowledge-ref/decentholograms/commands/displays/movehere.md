# /dh displays movehere

## Moves a Display Entity to your location

This command is only usable by a Player!

The `movehere` subcommand moves the specified Display Entity to your location.

## Permissions

-   `dh.command.displays`
-   `dh.command.displays.movehere`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `mvhr`

## Arguments

### `<name>`

Name of the Display entity to move to your location.
