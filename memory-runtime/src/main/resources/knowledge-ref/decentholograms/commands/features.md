# /dh features

## Allows the enabling/disabling of Features in the plugin

The `features` command provides a set of sub-commands that can be used to enable or disable Features in the plugin.

## Permissions

-   `dh.command.features`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `feature`
-   `f`

## Subcommands

-   ## [[/dh features disable|**/dh features disable**]]
    
    * * *
    
    Disables a Feature in the plugin
    
-   ## [[/dh features enable|**/dh features enable**]]
    
    * * *
    
    Enables a Feature in the plugin
    
-   ## [[/dh features info|**/dh features info**]]
    
    * * *
    
    Provides information about a specific feature.
    
-   ## [[/dh features list|**/dh features list**]]
    
    * * *
    
    Lists all available features
    
-   ## [[/dh features reload|**/dh features reload**]]
    
    * * *
    
    Reloads a feature
