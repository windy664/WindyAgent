# /dh version

## Displays the current plugin version

The `version` command prints the Version of DecentHolograms in the chat/console.  
This can be useful for reporting issues to share the version used.

## Permissions

-   `dh.command.version`

See the [[Command Permissions|Permissions]] page for more details.
