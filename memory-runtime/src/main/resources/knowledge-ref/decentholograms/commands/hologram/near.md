# /dh hologram near

## Lists all Holograms within a specific range

The `near` subcommand is used to list all Holograms within a specified range from your location.

Warning

This command is only usable by players.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.near`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<distance>`

Max distance in blocks from your location a Hologram should be to be listed.
