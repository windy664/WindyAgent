# /dh hologram update

## Forces a Hologram to update

The `update` subcommand forces a hologram to update by hiding and showing it again.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.update`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to update.
