# /dh hologram setpermission

## Sets a permission required to see the hologram

The `setpermission` subcommand allows to set a permission that players need to see the hologram.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.setpermission`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `permission`
-   `setperm`
-   `perm`

## Arguments

### `<hologram>`

Name of the Hologram to set the permission of.

### `[permission]`

Optional permission to set. Leave empty to remove any set permission.
