# /dh hologram info

## Displays general information about a Hologram

The `info` subcommand is used to see information about a Hologram.  
The displayed information contains the Hologram's name, location, whether its enabled or not, the permission set (if any), number of pages, its facing direction, down origin setting, update interval and range, display range and any Flags applied.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.info`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to retrieve info for.
