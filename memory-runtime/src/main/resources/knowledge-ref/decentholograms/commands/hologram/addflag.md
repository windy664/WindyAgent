# /dh hologram addflag

## Adds a Flag to a Hologram

The `addflag` subcommand allows to add a Flag to a Hologram.  
See the Flags page for available Flags.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.addflag`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to add a Flag to.

### `<flag>`

Name of the Flag to add.
