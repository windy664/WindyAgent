# /dh hologram removeflag

## Removes a Flag from a Hologram

The `removeflag` subcommand is used to remove a Flag from a Hologram.  
For a list of Flags, check the Flags page.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.removeflag`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to remove a flag from.

### `<flag>`

Name of the flag to remove.
