# /dh hologram enable

## Enables a Hologram

The `enable` subcommand is used to enable a Hologram.  
A enabled hologram will be visible to all players (unless permissions have been set for it).

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.enable`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `on`

## Arguments

### `<hologram>`

Name of the Hologram to enable.
