# /dh hologram delete

## Permanently deletes a hologram

The `delete` subcommand is used to permanently delete a Hologram.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.delete`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `del`
-   `remove`
-   `rem`

## Arguments

### `<hologram>`

Name of the Hologram to delete.
