# /dh hologram teleport

## Teleports you to a hologram

The `teleport` subcommand teleports you to the specified Hologram.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.teleport`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `tele`
-   `tp`

## Arguments

### `<hologram>`

Name of the Hologram to teleport to.
