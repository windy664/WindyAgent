# /dh hologram movehere

## Moves a Hologram to your current location

The `movehere` subcommand is used to move a Hologram to your current location.

Warning

This subcommand is only usable by players.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.movehere`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `mvhr`

## Arguments

### `<hologram>`

Name of the Hologram to move to your location.
