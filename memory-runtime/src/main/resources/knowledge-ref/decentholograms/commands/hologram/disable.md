# /dh hologram disable

## Disables a Hologram

The `disable` subcommand is used to disable a Hologram.  
Unlike the [[/dh hologram delete|`delete` subcommand]] does disabling a Hologram not delete it.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.disable`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `off`

## Arguments

### `<hologram>`

Name of the Hologram to disable.
