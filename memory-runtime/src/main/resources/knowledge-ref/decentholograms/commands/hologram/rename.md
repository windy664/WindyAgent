# /dh hologram rename

## Renames a Hologram

The `rename` subcommand is used to rename a Hologram.

## Permissions

-   `dh.command.hologram`
-   `dh.command.hologram.rename`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to rename.

### `<new_name>`

New name for the Hologram.
