# /dh displays

## Main command for display Entities features

The `displays` command is the main command used for everything related to DecentHolograms' Display Entities feature.

## Permissions

-   `dh.command.displays`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `display`
-   `d`

## Subcommands

-   ## [[/dh displays addline|**/dh displays addline**]]
    
    * * *
    
    Adds a Line to a Text Display Entity
    
-   ## [[/dh displays attribute|**/dh displays attribute**]]
    
    * * *
    
    Sets or gets attributes
    
-   ## [[/dh displays center|**/dh displays center**]]
    
    * * *
    
    Centers the Display Entity
    
-   ## [[/dh displays clone|**/dh displays clone**]]
    
    * * *
    
    Clones a Display Entity
    
-   ## [[/dh displays create|**/dh displays create**]]
    
    * * *
    
    Creates a new Display Entity
    
-   ## [[/dh displays delete|**/dh displays delete**]]
    
    * * *
    
    Deletes a Display Entity
    
-   ## [[/dh displays disable|**/dh displays disable**]]
    
    * * *
    
    Disables a Display Entity
    
-   ## [[/dh displays display-range|**/dh displays display-range**]]
    
    * * *
    
    Sets the range a Display Entity should be visible from
    
-   ## [[/dh displays enable|**/dh displays enable**]]
    
    * * *
    
    Enables a Display Entity
    
-   ## [[/dh displays help|**/dh displays help**]]
    
    * * *
    
    Show help about display commands
    
-   ## [[/dh displays insertline|**/dh displays insertline**]]
    
    * * *
    
    Inserts a new line into a Text Display Entity
    
-   ## [[/dh displays list|**/dh displays list**]]
    
    * * *
    
    Lists all currently created Display Entities
    
-   ## [[/dh displays list-attributes|**/dh displays list-attributes**]]
    
    * * *
    
    Lists all configured Attributes for a Display Entity
    
-   ## [[/dh displays move|**/dh displays move**]]
    
    * * *
    
    Moves the Display Entity
    
-   ## [[/dh displays movehere|**/dh displays movehere**]]
    
    * * *
    
    Moves a Display Entity to your location
    
-   ## [[/dh displays nearby|**/dh displays nearby**]]
    
    * * *
    
    Lists all nearby Displa Entities
    
-   ## [[/dh displays removeline|**/dh displays removeline**]]
    
    * * *
    
    Removes a Line from a Text Display Entity
    
-   ## [[/dh displays rename|**/dh displays rename**]]
    
    * * *
    
    Renames a Display Entity
    
-   ## [[/dh displays reset-attribute|**/dh displays reset-attribute**]]
    
    * * *
    
    Resets the provided attribute to its default value
    
-   ## [[/dh displays set-block|**/dh displays set-block**]]
    
    * * *
    
    Set the displayed block in a Block Display
    
-   ## [[/dh displays set-facing|**/dh displays set-facing**]]
    
    * * *
    
    Sets the facing angle of the Display Entity
    
-   ## [[/dh displays set-item|**/dh displays set-item**]]
    
    * * *
    
    Sets the item to display
    
-   ## [[/dh displays setline|**/dh displays setline**]]
    
    * * *
    
    Sets a line in the Text Display Entity
    
-   ## [[/dh displays swap-lines|**/dh displays swap-lines**]]
    
    * * *
    
    Swaps two lines in a Text Display Entity
    
-   ## [[/dh displays teleport|**/dh displays teleport**]]
    
    * * *
    
    Teleports you to the Display Entity
    
-   ## [[/dh displays update-interval|**/dh displays update-interval**]]
    
    * * *
    
    Changes the Update Interval of a Display Entity
