# /dh features enable

## Enables a Feature in the plugin

The `enable` subcommand can be used to enable a feature in DecentHolograms.

## Permissions

-   `dh.command.features`
-   `dh.command.features.enable`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `on`

## Arguments

### `<feature>`

Name of the feature that should be enabled.
