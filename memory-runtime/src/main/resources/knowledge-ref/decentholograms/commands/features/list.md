# /dh features list

## Lists all available features

The `list` subcommand lists all currently available Features in DecentHolograms.  
The list displays the Feature name and its current status (enabled/disabled).

## Permissions

-   `dh.command.features`
-   `dh.command.features.list`

See the [[Command Permissions|Permissions]] page for more details.
