# /dh features info

## Provides information about a specific feature.

The `info` subcommand provides information about a specified feature.  
The information includes its name, a description of what it is and does and its current status (enabled/disabled).

## Permissions

-   `dh.command.features`
-   `dh.command.features.info`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<feature>`

Name of the feature to display information of.
