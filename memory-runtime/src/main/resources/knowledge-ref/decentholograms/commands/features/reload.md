# /dh features reload

## Reloads a feature

The `reload` subcommand allows to reload a specific feature.

## Permissions

-   `dh.command.features`
-   `dh.command.features.reload`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<feature>`

Name of the feature to reload.
