# /dh features disable

## Disables a Feature in the plugin

The `disable` subcommand can be used to disable a feature in DecentHolograms.

## Permissions

-   `dh.command.features`
-   `dh.command.features.disable`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `off`

## Arguments

### `<feature>`

Name of the feature that should be disabled.
