# /dh lines swap

## Swaps two lines

The `swap` subcommand switches places of two lines.

## Permissions

-   `dh.command.lines`
-   `dh.command.lines.swap`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to swap lines of.

### `<page>`

Page in the Hologram to swap lines of.

### `<line1>`

Line number of the first line to swap.

### `<line2>`

Line number of the second line to swap.
