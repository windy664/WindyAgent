# /dh lines remove

## Deletes a Hologram Line

The `remove` subcommand removes a Line from the Hologram.

## Permissions

-   `dh.command.lines`
-   `dh.command.lines.remove`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `del`
-   `delete`
-   `rem`

## Arguments

### `<hologram>`

Name of the Hologram to remove a line from.

### `<page>`

Page in the Hologram to remove a line from.

### `<line>`

Line number to remove.
