# /dh lines removeflag

## Removes a Flag from a line

The `removeflag` subcommand removes a Flag from a Hologram Line.

## Permissions

-   `dh.command.lines`
-   `dh.command.lines.removeflag`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to remove a Flag from.

### `<page>`

Page in the Hologram to remove a Flag from.

### `<line>`

Line number to remove a flag from.

### `<flag>`

The Flag to remove from the Line.
