# /dh lines set

## Set the content of a Line

The `set` subcommand is used to set the Content of a specific Hologram Line.

## Permissions

-   `dh.command.lines`
-   `dh.command.lines.set`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to set the line content of.

### `<page>`

Page in the Hologram to set the line content of.

### `<line>`

Line number to set the line content.

### `<content>`

The new Line content to use.
