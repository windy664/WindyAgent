# /dh lines info

## Retrieves info about a Hologram Line

The `info` subcommand displays information about a Hologram Line.  
Information includes the Hologram name, the index, location, content, height and the X, Y and Z offset.

## Permissions

-   `dh.command.lines`
-   `dh.command.lines.info`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to get line info from.

### `<page>`

Page in the Hologram to get line info from.

### `<line>`

Line number to get info from.
