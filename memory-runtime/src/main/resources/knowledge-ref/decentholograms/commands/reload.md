# /dh reload

## Reloads the plugin and its holograms

The `reload` command performs a reload of DecentHolograms.  
This will refresh the general plugin settings, custom animations and all saved Holograms.

Note for developers

This command will trigger a [[DecentHologramsReloadEvent|`DecentHologramsReloadEvent`]]

## Permissions

-   `dh.command.reload`

See the [[Command Permissions|Permissions]] page for more details.
