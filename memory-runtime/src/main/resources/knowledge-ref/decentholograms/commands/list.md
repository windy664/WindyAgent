# /dh list

## Lists all currently created Holograms

The `list` command allows you to see all currently created Holograms, including their exact location.

## Permissions

-   `dh.command.list`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `[page]`

Optional page number to open the list at.
