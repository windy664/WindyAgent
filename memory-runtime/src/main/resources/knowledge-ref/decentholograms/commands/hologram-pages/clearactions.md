# /dh pages clearactions

## Removes all actions for a click type from a page

The `clearactions` subcommand removes all currently set Actions for a Click Type in a Hologram page.

## Permissions

-   `dh.command.pages`
-   `dh.command.pages.clearactions`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to clear a page's actions from.

### `<page>`

Page in the Hologram to clear actions from.

### `<click_type>`

The Click Type to remove all actions for.
