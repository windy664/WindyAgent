# /dh pages addaction

## Adds an action to a Hologram Page

The `addaction` subcommand adds a new Action to a Hologram Page for the specified Click Type.

## Permissions

-   `dh.command.pages`
-   `dh.command.pages.addaction`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to add a page's action to.

### `<page>`

Page in the Hologram to add a new action to.

### `<click_type>`

The Click Type for the action.

### `<action>`

The Action to add. See the Actions page for a list of available options.
