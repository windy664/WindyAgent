# /dh pages removeaction

## Removes an Action from a Hologram Page

The `removeaction` subcommand removes an action of the specified click type from a Hologram Page.

## Permissions

-   `dh.command.pages`
-   `dh.command.pages.removeaction`

See the [[Command Permissions|Permissions]] page for more details.

## Aliases

-   `remaction`

## Arguments

### `<hologram>`

Name of the Hologram to remove a page's action from.

### `<page>`

Page in the Hologram to remove an action from.

### `<click_type>`

Click Type from which to remove an action.

### `<index>`

Position of the action in the list to remove.
