# /dh pages actions

## Shows Click actions of a page

The `actions` subcommand allows to see Click actions configured for a specific Click type.

## Permissions

-   `dh.command.pages`
-   `dh.command.pages.actions`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to see a page's click actions.

### `<page>`

Page in the Hologram to see its click actions.

### `<click_type>`

The Click Type to list actions for.

### `[list_page]`

Optional Page number to show the list of.
