# /dh pages swap

## Swaps places of two Hologram pages

The `swap` subcommand switches the positions of two hologram pages.

## Permissions

-   `dh.command.pages`
-   `dh.command.pages.swap`

See the [[Command Permissions|Permissions]] page for more details.

## Arguments

### `<hologram>`

Name of the Hologram to swap pages in.

### `<page1>`

First page to swap.

### `<page2>`

Second page to swap with the first one.
