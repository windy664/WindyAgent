# lang.yml

## The lang.yml file

The `lang.yml` file is a file automatically created on the plugins first start.  
It gets populated with the individual messages the plugin can send as a response to different commands and actions.

You can edit this file to change the messages used within the plugin, i.e. to translate it into a different language.
