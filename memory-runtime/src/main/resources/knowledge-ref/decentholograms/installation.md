# First Install

## How to properly install DecentHolograms

This page will cover your first steps in installing DecentHolograms.

The installation is straight forward: Simply download the plugin from the platform of your choice (Modrinth or Spigot), stop your Server, upload the Jar to the `plugins` folder and start the server again.  
The plugin should generate a `DecentHolograms` folder containing all the Configuration files you may want to edit at a later point.

## Dependencies

DecentHolograms has no required dependencies, but supports the following optional ones:

### PlaceholderAPI

Should the PlaceholderAPI plugin be present and enabled, will DecentHolograms enable support for Placeholders from it.

### HeadDatabase

Should the HeadDatabase plugin be present and enabled, will DecentHolograms enable support for using `HEADDATABASE_<id>` in Head names (i.e. `#ICON: PLAYER_HEAD (HEADDATABASE_1)`).
