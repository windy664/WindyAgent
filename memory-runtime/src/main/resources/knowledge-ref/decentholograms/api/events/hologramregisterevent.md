# HologramRegisterEvent

## Extends DecentHologramsEvent

New since 2.9.6

This event is called whenever a Hologram is registered through `HologramManager#register(Hologram)`.

## Methods

| Method | Description |
| --- | --- |
| `getHologram()` | Gets the Hologram that got registered. |
