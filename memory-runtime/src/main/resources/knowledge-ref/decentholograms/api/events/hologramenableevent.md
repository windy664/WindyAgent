# HologramEnableEvent

## Extends DecentHologramsEvent

New since 2.9.6

This event is called whenever a Hologram gets enabled through its `enable()` method.

## Methods

| Method | Description |
| --- | --- |
| `getHologram()` | Gets the Hologram that got enabled. |
