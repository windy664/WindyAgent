# HologramDisableEvent

## Extends DecentHologramsEvent

New since 2.9.6

This event is called whenever a Hologram gets disabled through its `disable(DisableCause)` method.

## Methods

| Method | Description |
| --- | --- |
| `getHologram()` | Gets the Hologram that got disabled. |
