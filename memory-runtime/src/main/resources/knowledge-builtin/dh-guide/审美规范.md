# 全息审美规范

给「悬浮字」定一套克制、统一的默认审美。**AI 创建全息前应先检索本篇，按此组织内容，而不是随手堆颜色。** 服主可直接改本篇调整全服风格。

## 三条铁律
1. **少即是好**：3~5 行为宜，单行别太长（≈12 汉字/24 字符内），太长在世界里发虚。
2. **配色不超三色**：一个主题色（强调）+ 白/浅灰（正文）+ 深灰（分隔/弱化）。避免一行里彩虹色。
3. **有层次**：标题行加粗高亮，正文行朴素，分隔线弱化——靠明暗与粗细拉开主次，而非靠更多颜色。

## 结构模板（从上到下）
```
§8§m        §r §<主题色>§l 标 题 §r §8§m        §r   ← 标题：粗体+主题色，两侧删除线做横线
§7副标题 / 一句话说明                              ← 正文：浅灰，克制
§8▬▬▬▬▬▬▬▬▬▬▬                                    ← 分隔线：深灰，弱化
§f正文要点一                                        ← 白色正文
§f正文要点二
```

## 配色速查（§码）
- 主题色任选其一并全局统一：`§b`科技蓝 / `§a`自然绿 / `§d`活力粉 / `§6`温暖金 / `§c`警示红。
- 正文：`§f`白（主要）、`§7`浅灰（次要）。
- 弱化/分隔：`§8`深灰。
- 样式：`§l`粗体（只给标题）、`§o`斜体（少量点缀，如署名），慎用 `§k`（乱码，仅特效场景）。

## 符号面板（点缀分隔，别过量）
- 横分隔线：`§8▬▬▬▬▬▬▬`、`§8§m       §r`（删除线撑出的实线）、`§7━━━━━━`
- 装饰/项目符：`✦ ❖ ◆ ➤ ★ » « ✔ ✖ ● ○ │ ⬥`
- 每个全息挑 1~2 种符号贯穿，别一行一个花样。

## 三个成品模板

**出生点欢迎（主题蓝）**
```
§8§m       §r §b§l欢 迎 回 来 §r §8§m       §r
§7%player_name%，很高兴见到你
§8▬▬▬▬▬▬▬▬▬▬
§f在线 §b%server_online% §f人  ·  输入 §b/help §f看指令
```

**商店牌（主题金）**
```
§6§l✦ 幸 运 商 店 ✦
§8▬▬▬▬▬▬▬▬
§f右键此处打开
§7限时折扣进行中
```

**规则/警示牌（主题红）**
```
§c§l⚠ 保 护 区 ⚠
§8▬▬▬▬▬▬▬▬
§f此处禁止破坏 / PvP
§7违者踢出
```

## 落地提示
- 有 PlaceholderAPI 就用 `%player_name%`、`%server_online%` 等做动态内容，让牌子"活"起来。
- 标题两侧的 `§m ... §r` 会以删除线渲染成横线，是零依赖做"实心分隔"的常用手法。
- 定完风格把主题色写进你服的知识库覆盖本篇，AI 之后就按你的色走。
